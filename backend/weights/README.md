# YOLO Model Weights

Place your trained leaf disease detection model file `leaf_disease.pt` in this directory.

If you don't have a trained model, the app will use dummy data for demonstration purposes.

To train your own model:
1. Prepare a dataset of leaf disease images
2. Use YOLOv8 training scripts from Ultralytics
3. Save the trained weights as `leaf_disease.pt`
